# Features
- Volume and music playback control as default
- 12 Cherry MX compatable mechanical keys
- An OLED that (hopefully) displays your current audio playing (TBD) or does whatever else you want it to display
- A frog shaped case
- Direct connection to PC via USB-C for reduced latency
- A delete key in the center to smash to your hearts content
